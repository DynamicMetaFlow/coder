# Coder 

New coders can craft small projects in HTML, CSS, and Javascript, right from the web browser.

http://goo.gl/coder

### MANUAL INSTALL:

1. You need to have node.js and npm installed

2. In coder-base run "npm install" to download all the
   needed modules.
```bash
$ cd coder-base
$ npm install
```


3. Start Coder
```bash
$ cd coder-base
$ node localserver.js
```
